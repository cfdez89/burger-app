# burger-app
This project was created to learn the basics of React.

by [Carlos Fdez]

## Getting Started

1. Clone this repository

   ```bash
   git clone https://github.com/cfdez89/burger-app.git
   cd burger-app
   ```

1. Install the npm packages

   ```bash
   npm install
   ```

1. Run the app!

   ```bash
   npm start
   ```
